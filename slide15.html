15 / 18
Issue

Asynchronous Bugs

Handling async/await logic led to unhandled promise rejections and silent failures without stack traces.

Mitigation

Structured Error Handling

Implemented try-catch blocks in controllers and a centralized global error handler middleware.

Issue

Middleware Chaining

Incorrect order of middleware caused request bodies to be undefined (parsing issues) or auth checks to fail.

Mitigation

Pipeline Visualization

Mapped out the request flow and utilized app.use() strictly for parsing before route definitions.

Issue

CORS & Data Validation

Cross-Origin Resource Sharing errors during frontend-backend integration and invalid data crashing the server.

Mitigation

Configuration & Validation

Configured cors package for specific origins and added input validation to sanitise requests.

Issue

Performance & N+1 Queries

Fetching related data (tasks per user) inside loops caused excessive database calls and slow response times.

Mitigation

Optimized Joins

Replaced loops with single SQL JOIN queries and used EXPLAIN to verify indexing usage.
