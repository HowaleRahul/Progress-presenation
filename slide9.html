Create Tasks

Users can add new tasks with titles and descriptions. Implements input validation to prevent empty entries.

Update existing task information via PUT requests. Features modal-based editing for better UX.

Task Completion

Toggle task status with visual feedback (strikethrough styling). Updates database status field.

Delete Tasks

Remove unwanted tasks securely. Includes a confirmation step before executing the DELETE request.
